# 🐛 DefectFlow - 프리미엄 결함 관리 시스템

**버전**: 2.0.0 (Cloud Native)  
**핵심 가치**: 보안, 실시간 데이터 동기화, 엔터프라이즈 워크플로우

DefectFlow는 전문 QA 팀을 위해 설계된 고성능 프리미엄 결함 관리 시스템입니다. SPA(Single Page Application)의 성능과 Supabase 클라우드 백엔드의 강력함을 결합하여 최적의 경험을 제공합니다.

## 📁 아키텍처 및 기술 스택
- **프론트엔드**: 단일 페이지 애플리케이션 (HTML5, Vanilla JS, Premium CSS3)
- **백엔드 (BaaS)**: [Supabase](https://supabase.com/) (인증, PostgreSQL, 스토리지)
- **암호화**: `bcryptjs`를 통한 클라이언트 사이드 BCrypt 암호화 (엔터프라이즈급 보안)
- **연동 기능**: `html2canvas`를 활용한 실시간 화면 캡처 위젯

## 🛡 보안 및 인증
- **보안 로그인**: 모든 페이지 접근 시 인증 필수.
- **비밀번호 보호**: 비밀번호는 절대 평문으로 저장되거나 전송되지 않습니다.
- **세션 관리**: LocalStorage를 통한 지속적 세션 및 로그인 후 이전 페이지 자동 복구.
- **역할 기반 권한(RBAC)**: 테스터, 조치자(개발자), 관리자별 맞춤형 화면 제공.

## 🚀 주요 기능
- **스마트 대시보드**: 결함 상태 및 통계를 직관적인 차트와 실시간 지표로 시각화.
- **테스트 벤치 통합**: 결함 재현을 위한 전용 샌드박스와 원클릭 리포팅 위젯 지원.
- **스토리지 동기화**: 결함 스크린샷은 Supabase Storage에 자동 업로드되어 안전하게 보관됩니다.
- **한국 표준시(KST) 지원**: 모든 타임스탬프는 한국 시간(UTC+9)을 기준으로 자동 처리됩니다.

## 🛠 프로젝트 표준 (Smart Working)
1. **깊이 있는 영향도 분석 (Impact Analysis)**: 모든 수정 사항은 프로젝트 전반에 미치는 부작용을 사전에 검증합니다.
2. **선제적 문제 대응 (Proactive Debugging)**: 라이브러리나 환경 오류 가능성을 미리 파악하여 선제적으로 대응합니다.
3. **'First-Time-Right' (한 번에 완벽하게)**: 반복 수정을 지양하고, 첫 배포 시 완벽한 결과로 사용자님의 시간을 보호합니다.
