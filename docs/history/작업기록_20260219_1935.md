## 주요 작업 사항

### 1. 스마트 작업 3대 핵심 요구사항 (USER Requirements)
- **깊이 있는 영향도 분석**: 수정 전 전역적인 사이드 이펙트 분석.
- **선제적 문제 대응**: 잠재적 오류 요소를 미리 파악하여 방어 코드 적용.
- **'First-Time-Right'**: 한 번의 배포로 완벽한 결과 도출하여 사용자 시간 보호.

### 2. 기술 연동 및 보안 강화
- **연동 위젯 고도화**: 외부 사이트에서 호출 시 실시간 스크린샷 캡처 및 로그인 세션 복구 흐름 가이드화.
- **DB 보안 설계**: 비밀번호 BCrypt 해싱 저장 및 초기 비밀번호 변경 플래그(`needs_pw_reset`) 도입.
- **인증 리다이렉트**: 비로그인 시 기존 요청 페이지(`returnTo`) 정보를 유지하여 로그인 후 즉시 복구되는 UX 설계 반영.

### 3. PMO 전략 프롬프트 최적화
- **정치/재무 리스크 관리**: 지연 사유 실명 트래킹, 의사결정 지연 비용(Cost of Delay) 계산 등 C-Level 보고용 지표 추가.
- **독한 질문 커맨드**: "진행률 허수 발라내기", "운영 거부 기능 식별" 등 PMO의 실무 통제 기능을 강화하는 명령 세트 추가.
